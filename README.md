
### 下棋AI算法
由于我们总可以保证人类这边下棋是下是最优的一步，所以自动任务（第四第五题）的算法使用 **min-max算法**，这是一种递归的算法，一种动态博弈的算法，一种假设人类也在采取最佳行动的情况下确定最佳行动的算法

需要注意的是 min-max 算法并不是最优的算法，因为当对手出现失误的时候，这个算法并没有发现漏洞的能力，可以运行```..test/test_policy.py```算法测试尝试。
